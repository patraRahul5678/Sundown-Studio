# 🌇 Sundown Studio Clone

A responsive **frontend clone of the Sundown Studio website**, built to practice modern web development concepts, smooth animations, and pixel-perfect UI design.  
This project focuses on recreating the look, feel, and interactions of the original website using clean and optimized code.

(https://patrarahul5678.github.io/Sundown-Studio/)

## 📌 Features

- 🎨 Pixel-perfect UI inspired by Sundown Studio
- 📱 Fully responsive design (mobile, tablet, desktop)
- ✨ Smooth scrolling and animations
- 🧩 Clean and reusable code structure
- ⚡ Fast performance and optimized assets

---

## 🛠️ Tech Stack

- **HTML5** – Semantic markup  
- **CSS3** – Modern styling & animations  
- **JavaScript (ES6+)** – Interactivity & logic  
- **Locomotive Scroll / GSAP** – Smooth scrolling & animations
